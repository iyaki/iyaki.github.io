Sobre este sitio

Hace algún tiempo compré en NIC Argentina el dominio iyaki.ar. El primer paso obvio luego de eso era montar una página web (y si sobraba tiempo configurar un servidor de correo electrónico).

Después de múltiples búsquedas en Google sobre servicios de hosting y demás terminé creando esta página utilizando:

  • Github Pages para el alojamiento web.
  • Cloudflare para el registro de los DNS y CDN.
  • Mis manos y el Visual Studio Code (con algunas extensiones) para escribir el código fuente de esta página.
  • Eleventy para pre-procesar el HTML de manera estática y poder reutilizar código.

Eso es todo. No hay paso de compilación o transpilación, no hay frameworks, ni siquiera bibliotecas de CSS o JS. (Post, explicando las razones de este cambio)

En esta primer versión de la página eso puede parecer una obviedad, pero mi intención es seguir desarrollandola de manera incremental y agregando, paulatinamente; estilos, funcionalidad y nuevas secciones.

Por el momento me centro, principalmente, en que el uso de los elementos HTML sea semánticamente adecuado y accesible.


Mi objetivo al crear esta página no es contar con un portfolio para agregar en mi CV sino poder compartir mis experiencias y vivencias.

Si algo de lo que ven les interesa; tienen sugerencias, criticas o cualquier comentario pueden escribirme a: me@iyaki.ar (finalmente no configure un servidor de correo electrónico pero sí un sistema de redirección y envío de e-mails utilizando los servicios de Cloudflare Email Routing y Zoho Mail, respectivamente).
