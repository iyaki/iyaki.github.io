iyaki

Generalista

Estos son algunos de mis proyectos más recientes. Muchos de ellos nacieron para ayudarme a simplificar tareas diarias y procesos complejos. Algunos me ayudaron a probar y aprender nuevas tecnologías y otros son experimentos para explorar formas distintas de explotar tecnologías o herramientas que ya conocía.

Proyectos

ralphex

Ralphex es un CLI specs-first para ejecutar loops de codificación iterativos contra CLIs de agentes externos como OpenCode, Claude Code y Cursor. Ofrece prompts integrados de plan y build, precedencia de configuración determinista y un modelo de comando simple que funciona bien en flujos de trabajo basados en repositorios.

reglint

RegLint es un linter basado en regex para repositorios de código fuente. Escanea archivos usando reglas definidas en YAML y emite salida en console, json o sarif para desarrollo local y pipelines de CI. (WIP, evolución de regex-pr-annotator)

devcontainers-features

Una colección de características para Devcontainers que facilitan la configuración de entornos de desarrollo consistentes y reproducibles.

daily-knowledge

Implementación web de random-knowledge, un pequeño script basado en RandomNote Web diseñado para trabajar sobre mi base de conocimiento en Notion.

content-curator

Código fuente de la aplicación que replica el contenido de mi base de conocimiento en Notion al recopilador de contenido curado disponible en curated.iyaki.ar/

skills-update

Acción de GitHub que mantiene actualizadas las skills de los agentes en tu repositorio con un solo comando, ejecutando la CLI de Skills de Vercel en CI.

web-archiver

Utilidad de línea de comandos para guardar páginas web desde un sitemap en el Web Archive (archive.org) con capturas completas (screenshots, enlaces externos).

simple-newsletter

Servicio gratuito que convierte Feeds RSS en Newsletters enviadas por e-mail, cualquiera puede utilizarlo.

El juego de la papa online

Juego en línea donde el objetivo es conectar números en orden sin que las líneas se crucen. Desarrollado basado en la idea de un amigo, con código disponible en repositorio privado.
